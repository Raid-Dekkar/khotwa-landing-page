# Khotwa Project

**Khotwa** is a digital scouting education initiative built around an **interactive Telegram bot** for Cubs in the Algerian Muslim Scouts ecosystem.  
It provides structured educational content, guided activities, and practical resources for leaders, children, and parents.

**Visit the website:** [Khotwa Project](https://raid-dekkar.github.io/khotwa-landing-page/)

---

## 📘 Project Description

Khotwa aims to make scouting education more accessible through a guided Telegram experience while preserving core scouting values.  
The bot content is organized into multiple thematic tracks covering spiritual, social, cognitive, physical, and practical development.

---

## ✨ Key Features

- **Interactive Telegram bot–centered experience**
- **Structured educational tracks** for Cubs and leaders
- **Reference-based content model** (built around the Khotwa guidebook)
- **Full Arabic RTL interface** for local audience communication
- **Responsive, mobile-friendly UI**
- **Clear onboarding journey** (Open Bot → Choose a Track → Interact)
- **Integration points for media and community channels**
